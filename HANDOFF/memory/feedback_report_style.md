분석 리포트에서 '소재 지형' 같은 부자연스러운 추상화 용어를 사용하지 않는다. 개념을 임의로 추상화하지 말고, 특성을 자연스러운 언어로 서술하는 데 집중한다.

**Why:** 이 매뉴얼은 '수능 국어를 이해하는 에이전트'를 학습시키기 위한 합성 데이터셋 제작에 사용된다. 에이전트가 임의 추상화 개념이 아니라 자연스럽고 직관적인 언어로 수능 지문을 이해할 수 있어야 한다.

**How to apply:**
- 섹션 제목도 자연스럽게 (예: "소재 지형" → "어떤 소재가 출제되는가")
- 예시 사용 시 세트 번호만 언급하지 말고, 해당 지문이나 문단 전문을 인용해야 한다
- 임의 조어 금지 (소재 지형, 개념 공백 추론 유도, 필연적 부착, 사고 유도 요소 등)
- 인지 부하, 독해 포인트 등 일반적 용어는 사용 가능
